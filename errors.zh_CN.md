## OCX 开发者接口错误列表 (API version 2)

REST API 错误代码


## 身份验证系列

| 参数        | 详细描述                                                     |
| ---------- | ----------------------------------------------------------- |
| 40100      |   令牌无效                                                    |
| 40102      |   签名无效                                                    |
| 40103      |   时间戳无效                                                   |
| 40104      |   时间戳重复                                                   |
| 40105      |   access_key 被禁用                                           |
| 40106      |   access_key 已过期                                           |
| 40107      |   权限不足                                                    |
